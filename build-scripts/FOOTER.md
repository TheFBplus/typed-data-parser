### For Developer:

clone repository

```bash
git clone https://github.com/TheFBplus/typed-data-parser.git
```

init development environment

```bash
npm install
or
npm run init
```

test

```bash
npm run test
```

build

```bash
npm run build
```

build api docs

```bash
npm run docs
```
